About Me

I perform IT troubleshooting and consulting duties at work, though deep down I am a cloud, virtualization and open source enthusiast.

I've dabbled in many programming languages and a number of frameworks over the years since I started programming around 2015, but my current (as of February 2022) focus is on cloud computing, as demonstrated by the ACA Cloud Computing certification and 10+ Clouders on Alibaba Cloud.

More recently, I have been writing articles on Dev.to related to cloud computing, with AWS as the reference platform, shown below.

Introduction to Infrastructure as Code with Terraform and Packer

This article offers a brief introduction to Infrastructure as Code (IaC), followed by a hands-on session with Terraform, HashiCorp's IaC solution: installing Terraform, provisioning an Amazon EC2 instance declaratively with HashiCorp Configuration Language (HCL) files, automatically running a script on first boot in an EC2 instance with user data. Near the end of the article, we also consider the drawbacks of running an automated setup script on every new EC2 instance, and address the issue with HashiCorp's image-building tool Packer.
